# Сборка электронной схемы и Arduino-скетч 

Проект **KeyLamp-Arduino** — это DIY лампа на базе Arduino, которая меняет цвет при переключении раскладки клавиатуры. Идея в том, чтобы визуально показывать активную раскладку (например, русский - английский) с помощью RGB-светодиода.  
Индикация реализована через Arduino и RGB LED, считывая состояние раскладки с компьютера.

## Что делает

- Индикатор активной раскладки клавиатуры с помощью цвета RGB-светодиода
- Простая реализация на Arduino

## Требуемое оборудование

Комплектующие:
- Arduino Nano
- RGB светодиод
- Резистор, соответствующий светодиоду (220–330 Ом для каждого цвета)
- USB-кабель для подключения к ПК
- (Опционально) Провода для соединений 
- (Опционально) макетная плата

Инструменты:
- USB-паяльник и припой для пайки

## Как работает

1. Arduino принимает данные о текущей раскладке с компьютера (через последовательный порт USB).
2. В зависимости от раскладки светодиоды меняют цвет: **Красный = Русский**, **Зелёный = Английский**.
3. Светодиод горит соответствующим цветом — визуальная индикация раскладки.

## Быстрый старт

### 1. Настройка

Скачайте к себе репозиторий:

```bash
git clone https://github.com/lshnls/keylamp-arduino.git
```
Откройте проект в Arduino IDE или VSC PlatformIO.

### 2. Сборка схемы arduino и светодиода.

Назначение пин Arduino:
Красный	D9
Зелёный	D10
Синий	D11
GND	GND

![arduino-with-led](img\arduino-with-led.png)

Подключите собранную схему к компьютеру по USB

### 3. Загрузка прошивки

Загрузите скетч в Arduino через IDE:
Выберите плату и порт.
Нажмите Upload.

### 4. Проверка работы


- Подключите собранную схему с Arduino к компьютеру и убедитесь, что светодиоды загораются.
- Откройте консоль и отправьте команды через Serial Monitor '1'<enter>, '2'<enter>,'3'<enter>,'9'<enter>. LED должен загореться последовательно разными цветами.

Совет: Если LED не меняет цвет, проверьте соединения, правильность пинов и настройки Serial на компьютере.
